# Apple Carbon Emissions Dashboard

This project analyzes Apple's greenhouse gas emissions using Power BI.

## Tools Used
- Power BI
- Data Visualization
- DAX

## Dashboard Features
- KPI cards for total, average, and max emissions
- Emission trend analysis by fiscal year
- Scope 1, Scope 2, Scope 3 emission distribution
- Category-level emission analysis
